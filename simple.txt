# Простий аналіз даних в R
# Створюємо вектор чисел
numbers <- c(10, 20, 30, 40, 50)

# Розраховуємо основні статистичні показники
mean_value <- mean(numbers)
median_value <- median(numbers)
sd_value <- sd(numbers)

# Виводимо результати
print(paste("Середнє значення:", mean_value))
print(paste("Медіана:", median_value))
print(paste("Стандартне відхилення:", sd_value))

Опис програми:
Цей код демонструє базові можливості мови R для статистичного аналізу.
Функціонал:
- Створення числового вектора
- Розрахунок середнього значення
- Розрахунок медіани
- Розрахунок стандартного відхилення
